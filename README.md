Описание проекта

Разработана мультиагентная система для проведения технического интервью, состоящая из двух агентов: Interviewer и Observer.

Interviewer — агент, который непосредственно взаимодействует с пользователем: задаёт вопросы на основе рекомендаций агента Observer, управляет ходом интервью и формирует итоговый вердикт для пользователя.

Observer — аналитический агент, который анализирует ответы пользователя, формирует рекомендации для агента Interviewer по дальнейшим вопросам, а также принимает решение о продолжении или остановке интервью.

Система предназначена для автоматизации процесса технических собеседований и повышения качества оценки кандидатов.
